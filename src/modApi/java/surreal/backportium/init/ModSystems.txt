Map Markers: Right click on a banner with map to add it as marker. Right click again to remove it. It shows color and name of the banner.
Swimming: Obvious.
Commands:
- Command Completion
- Local Coordinates '^': (?) Basically ~ but uses player orientation for facings.
- /bossbar: Creating boss bars.
- /data: changing nbts of entities, tile entities etc.
- /locate: Locating structures
- /scoreboard: Some changes
- /teleport facing: 'facing' value which allows the teleported entity to face an entity or location.
- /time: Added noon and midnight
